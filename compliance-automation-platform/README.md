# CMMC Compliance Automation Platform

## The Problem
Small businesses face a federal compliance framework (CMMC) they don't have the
resources to interpret, with few qualified assessors available and consulting costs
pricing most of them out entirely.

## The Solution
A self-service platform that translates CMMC requirements into actionable compliance
blueprints.

**Backend:** Supabase (PostgreSQL relational database with built-in authentication)
**AI Integration:** Claude API, with custom system prompt design for repeatable,
auditable compliance blueprint generation
**Payments & Analytics:** Stripe, PostHog
